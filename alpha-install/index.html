APK Installation Guide

⏱️ Takes about 2 minutes • Only needed once • Works on Android 7.0+
📥 Download Build

Then follow these steps:

1
Open the APK file
Tap the download link to open your Google Drive folder. You'll see the APK file - tap on it to start installation.
2
Tap "Install"
You'll see "Install this app?" with the FarmCash icon. Tap the "Install" button to continue.
3
Tap "Settings"
You'll see "Permission required" - your phone needs permission to install apps from Drive. Tap "Settings" to enable it.
4
Enable "Allow from this source"
Toggle the "Allow from this source" switch to ON. This lets Drive install apps - it's safe and you can turn it off later.
5
Go back and tap "Install" again
Tap the back button to return to the installer. You'll see the "Install" button again - tap it!
6
Handle Play Protect (optional)
Google Play Protect might recommend scanning the app. Let it scan!
7
Tap "Install" (final time!)
If you let Play Protect scan, it'll say "This app looks safe". Tap "Install" to proceed. The installation takes 10-20 seconds.
8
Tap "Open" and start testing!
Success! You'll see "App installed" with an Open button. Tap it to launch FarmCash and start the tutorial!
✅ You're all set!
